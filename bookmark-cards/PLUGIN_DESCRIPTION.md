# Bookmark Cards

> 把便利贴搬进 Obsidian 太治愈了！📚✨

---

## 一句话描述

**可视化书签管理插件，支持列表/卡片双视图，自动抓取网站封面和元数据。**

---

## 核心功能

✅ **双视图模式** - 列表/卡片一键切换
✅ **智能元数据抓取** - 自动获取标题、描述、封面
✅ **Markdown原生支持** - 支持标题、列表、Wiki-links
✅ **优雅交互体验** - 悬停动画、移动端适配
✅ **深色主题适配** - 完美融入Obsidian风格
✅ **元数据缓存** - 智能缓存，避免重复请求

---

## 快速开始

1. 安装插件
2. 创建收藏夹 Markdown 文件
3. 配置文件路径
4. 点击左侧边栏书本图标
